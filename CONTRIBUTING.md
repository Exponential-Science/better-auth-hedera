## How to Contribute
1. Fork the repository.
2. Create a branch:

   ```bash
   git checkout -b feature/my-change
   ```

3. Make your updates.
4. Commit with a sign-off (git commit -s -m "my change").
5. Open a Pull Request against main.

### Code Standards
- Follow existing formatting and linting.
- Add tests for new functionality.
- Include or update SPDX headers on all new files.
